Random Walkers Sandbox

This project explores the random walkers theme and allows for a variety of values to change that will affect how the walkers move.
It also allows the user to select an image and select the color pattern "picture" to draw the picture as the walkers move around the screen.
Click the canvas to add a walker at that location!
